Centralized Platform for Community Outreach

Developed a full-stack .NET application to centralize data, streamline outreach efforts, and provide comprehensive reporting on services for the unhoused population.

.NET / C# GIS / Mapping Reporting Project Management

The Problem

  • Critical data was siloed within a centralized work order management system, preventing effective collaboration between different outreach teams.
  • Outreach teams lacked real-time information on individual needs, service history, or last known locations.
  • Reporting to stakeholders was a manual, time-consuming process with a high risk of data inaccuracy.

The Solution

A secure, centralized .NET web application providing a single source of truth. The platform featured role-based access for different agencies, an integrated GIS component to visualize client locations, and a project management module to track outreach tasks and resource allocation.

The Process: A Multi-faceted Approach

The application was built using ASP.NET Core for the backend API and a responsive front-end. The data layer was built on a SQL Server database with carefully designed tables. Custom ETL processes were crafted to summarize and import data from various sources into the correct, standardized structures, ensuring data integrity and performance.

We integrated with ArcGIS services to provide a live map view of client check-ins and known encampment locations, giving outreach teams critical spatial awareness. For reporting, we developed a custom dashboard using D3.js to create dynamic, interactive graphs. These visualizations were powered by optimized, custom-written SQL queries to provide on-demand insights for city officials and funding partners.

A custom project management module was developed to allow team leaders to create tasks (e.g., "Welfare Check," "Supply Drop"), assign them to outreach workers, and track their status in real-time. This feature replaced a chaotic system of emails and phone calls, dramatically improving operational efficiency.

Key Takeaway

This project highlighted the power of technology to drive social impact. By integrating disparate systems into a single, user-friendly platform, we enabled a more efficient, data-driven, and compassionate response to a critical community issue.
